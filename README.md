# Guess My Number

**Guess My Number** is a simple web game built with JavaScript, HTML, and CSS.

## Features

- Generates a random number to guess
- Lets the user input a number
- Displays hints like "Too low" or "Too high" when the guess is incorrect
- Tracks current score and best score
- "Try Again" button to restart the game

## Installation

1. Clone this repository:  
   `git clone https://github.com/cynhlfn/guess_my_number_js.git`

2. Open `index.html` in your web browser.

## Technologies Used

- HTML
- CSS
- JavaScript
